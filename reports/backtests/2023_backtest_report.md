## 2. Backtest Protocol

The backtest follows the established holdout protocol:

- **Training data**: 2021–2024 seasons (Elo ratings, feature computation, Platt calibration)
- **Analysis period**: 2023 season(s)
- **Chronological**: Elo ratings and rolling features updated sequentially within 2021–2024
- **Leakage controls**: Neutral-site games excluded; ties excluded via model_eligibility flag; no future data used in any feature
- **Model fixed**: Parameters set once by rolling-origin validation + forward selection; no tuning on held-out data
- **Predictions**: Generated by `make predict-incumbent` using `generate_incumbent_predictions()`

## 14. Key Failure Modes

Based on subgroup analysis and extreme games:

1. **QB-change games**: log loss 0.6312 vs 0.6582 for non-change games (gap -0.0270). This is the model's largest systematic weakness — Elo undershoots when QBs change due to injury or benching.
2. **High confidence (>0.80)**: log loss 0.3870, n=15. The model is overconfident in longshot away teams despite high predicted home win probabilities.
3. **Monday night games**: log loss 0.8508 vs Sunday 0.6383 (small sample).
4. **Early season (W1-4)**: log loss 0.6387 vs 0.6839 late season — the model improves with more in-season data.

## 15. What the Model Understands

- Home field advantage: correct advantage direction in most games
- Team strength differentials: well-calibrated probabilities for mid-range favorites (0.55–0.75)
- Late-season form: rolling_mov_3 captures recent performance trends
- QB changes: qb_changed feature helps but does not fully resolve the gap
- Divisional games: comparable performance to non-divisional
- Postseason: lower error than regular season (smaller sample)

## 16. What the Model Still Misses

- **QB-change timing**: the binary qb_changed flag does not capture the magnitude of QB quality drop
- **Extreme confidence calibration**: >0.80 predictions are overconfident
- **Early-season uncertainty**: weeks 1–4 have higher error (less in-season data for Elo)
- **Monday night variability**: small-sample volatility in primetime away games
- **Market-relative information gap**: market (0.6090) beats incumbent (0.6262) by 0.0172 log loss

## 17. Incumbent Validity

This backtest is an in-training diagnostic (season 2023 was part of training data).
Log loss cannot be compared to the 2025 holdout benchmark directly.

## 18. Recommended Next Experiments

1. **QB-change magnitude**: Replace binary qb_changed with a continuous measure of QB quality delta (e.g., EPA/play difference between starter and backup)
2. **Leverage opening market lines**: Opening lines would give a fairer pregame market comparison and reveal where Elo can win on information advantage
3. **Calibration by era**: Early-season (W1–4) Platt scaling vs rest-of-season could reduce early-season error
4. **High-confidence regularization**: Temperature scaling or Platt shrinkage for the >0.80 bucket to reduce overconfidence
5. **Add more seasons**: Expanding training data (pre-2021) may improve estimates for QB-change and early-season regimes, if leakage can be avoided
