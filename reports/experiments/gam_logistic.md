# GAM/Spline Logistic Experiment

## Research Question

Does a nonlinear (spline) transformation of `elo_prob` improve the v3.0.0 Frozen QB Overlay champion?

## Architecture

```
For each rolling-origin fold:
  1. Compute Elo features chronologically
  2. Build X = [SplineTransformer(elo_prob), qb_changed, mov_3,
                rest_diff, is_dome, div_game]
  3. Fit LogisticRegression on train seasons only
  4. Apply frozen QB overlay (v3.0.0 params) on top
  5. Score on validation season
```
