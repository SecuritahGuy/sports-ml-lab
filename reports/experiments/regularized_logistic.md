# Regularized Logistic Meta-Model Experiment

## Research Question

Can a regularized logistic regression on the v3.0.0 incumbent logit plus additional pregame features improve on the v3.0.0 Frozen QB Overlay champion?

## Architecture

```
For each rolling-origin fold:
  1. Fit fold-safe incumbent (Platt + QB overlay) on train seasons
  2. Compute incumbent_logit = logit(incumbent_prob)
  3. Build meta-features = [incumbent_logit] + rest_diff + is_dome + div_game + week_sin + week_cos + game_type_is_playoff
  4. Fit logistic regression meta-model (tuning C + penalty) on train
  5. Score on validation season
```

## Meta-Features

| Feature | Description |
|---------|-------------|
| incumbent_logit | logit(v3.0.0 probability) — the incumbent signal |
| rest_diff | home_rest - away_rest (rest advantage) |
| is_dome | 1 if game is in domed stadium |
| div_game | 1 if division matchup |
| week_sin | sin(2π * week / 18) — cyclic week encoding |
| week_cos | cos(2π * week / 18) — cyclic week encoding |
| game_type_is_playoff | 1 if playoff game |
