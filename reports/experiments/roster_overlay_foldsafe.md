# Fold-Safe Roster Overlay Experiment

## Research Question

Do position-group availability overlays (OL, skill, front, LB, coverage) improve on the frozen QB overlay incumbent (holdout LL 0.6200)?

## Methodology

For each position group, a depletion score is computed from injury report OUT counts:

```
depletion = min(OUT_count / typical_depth, 1.0)
gate = max(home_depletion, away_depletion) > threshold
net = home_depletion - away_depletion
overlay = gamma * net * ln(10)/400
```

The overlay is applied in logit space and gated. Non-gated games are identical to the incumbent.

## Incumbent

v3.0.0: Standard Elo + qb_changed + rolling_mov_3 + Platt + frozen QB overlay. Holdout LL: 0.6259.

## Decision

**❌ REJECTED**

No roster overlay beats the incumbent on both validation and holdout.

## Failure Modes

1. **Injury data coverage**: Injury reports may miss some players or teams.
2. **OUT counts ≠ impact**: Not all OUT players are equal (starter vs backup).
3. **Coarse measure**: Availability is a simple count, not a skill rating.
4. **No per-player ratings**: Unlike the QB overlay, we don't have Bayesian-shrunken performance ratings for non-QB positions.
