# Expanded Elo Spine + Frozen QB Overlay

## Research Question

Can a better base Elo spine improve the v3.0.0 Frozen QB Overlay champion? The QB overlay improved the top layer. Now we test whether improving the base Elo probability underneath yields further gains.

## Architecture

```
base Elo (swept: K, HFA, reg, decay)
→ fold-safe Platt on [elo_prob, qb_changed, rolling_mov_3]
→ frozen QB overlay (v3.0.0 champion, fixed)
→ validation/holdout comparison
```

All candidates use the **same frozen QB overlay** (gate: H. changed OR starts<17, cap=40, gamma=1.0). Only the Elo base parameters change.

## Decision

**❌ REJECTED**

No candidate beats v3.0.0 on both validation and holdout.

The expanded Elo spine search did not find a better base model under the frozen QB overlay. The v3.0.0 champion remains the best known configuration.

## Takeaways

1. **The v3.0.0 Elo base (K=36, HFA=40, reg=0.1, decay=32) is robust** — no expanded Elo spine candidate beat it on both val and holdout.
2. **Platt scaling absorbs much of the base-Elo variation** — logistic calibration on [elo_prob, features] compresses differences in raw Elo accuracy.
3. **The QB overlay dominates** — the overlay's logit adjustment of up to ±40.0 elo points swamps the Elo spine differences.
