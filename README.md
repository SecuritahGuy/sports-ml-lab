# Sports ML Lab

This repository contains the code for the Sports ML Lab project.

## Structure

The project is organized as follows:
- `src/` - Source code files
- `tests/` - Test files
- `data/` - Data files and datasets
- `docs/` - Documentation files
- `config/` - Configuration files
